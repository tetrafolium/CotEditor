Syntax style information settings

The metadata of syntax style file can be set as follows:

Version The version of the syntax style.
Last modified The last modified date of the syntax style. We recommend to use the format YYYY-MM-DD.
Distribution URL The URL of a web page in which the syntax style is distributed.
Author The author of the syntax style.
License The license of the syntax style.
Description The additional information of the syntax style.

These infos might be useful if you want to share your own syntax style. You can leave fields also empty.
